# אתר שיווק דיגיטלי | Digital Marketing Website

אתר אישי קטן לשיווק דיגיטלי — דף בית, אודות, שירותים ובלוג. עברית כברירת מחדל עם תמיכה באנגלית.

---

## תוכן האתר

1. **דף בית (Landing)** — כותרת, תיאור וכפתורי פעולה  
2. **אודות** — הסיפור שלך בעולם הדיגיטל  
3. **שירותים** — מה את מציעה (אסטרטגיה, תוכן ו-SEO, רשתות, פרסום)  
4. **בלוג** — כרטיסיות לפוסטים (טוב ל-SEO ולסמכות)

---

## איך לצפות באתר

### אפשרות 1: פתיחה מקומית (ללא שרת)

- פתחי את הקובץ `index.html` בדפדפן (לחיצה כפולה או גרירה לחלון הדפדפן).  
- כתובת תופיע כ־`file:///...` — זה תקין.  
- אם משהו לא עובד (למשל שמירת שפה), השתמשי באפשרות 2.

### אפשרות 2: שרת מקומי (מומלץ)

בטרמינל, מתוך תיקיית הפרויקט:

```bash
# אם מותקן Python 3:
python3 -m http.server 8080

# או עם npx (אם יש Node):
npx serve -p 8080
```

ואז בדפדפן: **http://localhost:8080**

## איך לקבל כתובת URL אמיתית (אונליין)

כדי שהאתר יהיה זמין באינטרנט עם לינק קבוע:

1. **GitHub Pages (חינם)**  
   - העלי את התיקייה ל־repository ב־GitHub.  
   - Settings → Pages → Source: main branch.  
   - האתר יופיע בכתובת: `https://<username>.github.io/<repo-name>/`

2. **Netlify (חינם)**  
   - גררי את תיקיית הפרויקט ל־[netlify.com/drop](https://app.netlify.com/drop).  
   - תקבלי לינק כמו: `https://random-name.netlify.app`

3. **Vercel**  
   - חיבור ל־Git והעלאה — תקבלי דומיין חינמי.

## איך לערוך את האתר

- **טקסטים (עברית/אנגלית):** בקובץ `js/main.js` — אובייקט `translations` (מפתחות `he` ו־`en`).  
- **מבנה ותוכן:** בקובץ `index.html` — כותרות, פסקאות, לינקים.  
- **עיצוב:** בקובץ `css/style.css` — צבעים, גופנים, רווחים.  

אין צורך בבנייה (build) — כל שינוי נשמר ונראה ברענון הדף.
